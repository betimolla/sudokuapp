# Sudoku App

A simple, responsive Sudoku game. This README provides a short overview, installation and usage instructions, and basic gameplay rules.

## Features

- **Generate puzzles:** New games with selectable difficulty (Easy / Medium / Hard).
- **Real-time validation:** Incorrect entries are highlighted immediately.
- **Timer:** Track how long it takes to solve a puzzle.
- **Responsive design:** Usable on desktop and mobile devices.

## Tech stack

- Core: React (replace with your stack if different)
- Styling: CSS

## Installation

1. Clone or download the repository.
2. Open a terminal in the project folder.
3. Install dependencies and start the development server:

```bash
npm install
npm start
```

Replace `npm` with `yarn` if you use Yarn.
